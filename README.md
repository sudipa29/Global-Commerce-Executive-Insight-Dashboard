## 🚀 Project Overview
An interactive Power BI dashboard designed to analyze global sales, profits, and regional market trends. Built on a optimized **Star Schema** data model, this project transforms raw commerce data into actionable business intelligence for executive-level decision-making.

---

## 🛠️ Key Features & Tech Stack
* **Data Modeling:** Built using a Star Schema with robust relationships between fact and dimension tables.
* **ETL Processing:** Cleaned and transformed raw data using **Power Query** and **SQL**.
* **Advanced DAX:** Authored custom measures for key performance indicators (KPIs) including *Total Sales*, *Profit Margin*, and *MoM Growth*.
* **Interactive Visuals:** Features advanced geographical maps, dynamic trend analysis, and granular drill-downs for deep-dive exploration.

---

## 📂 How to Use This Repository
1. Download the `.pbix` file from this repository.
2. Open it using **Power BI Desktop** to explore the local data model and interactive features.
